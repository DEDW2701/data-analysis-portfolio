# Data Analysis Portfolio

A static portfolio website showcasing data analysis projects built with Python, Pandas, Matplotlib, Seaborn, and Scikit-learn.

## Projects

| Project | Tools | Description |
|---|---|---|
| [Retail Sales Analysis](projects/retail-sales/analysis.py) | Pandas, Matplotlib, SQL | 2-year retail revenue trend, category & regional breakdown |
| [Supply Chain Lead Time Variance](projects/supply-chain/analysis.py) | Pandas, Seaborn, NumPy | Lead time variance across 8 suppliers, reliability scorecard, delay root causes |
| [Customer Churn Prediction](projects/customer-churn/analysis.py) | Scikit-learn, Seaborn | Logistic regression + random forest, 87% accuracy |
| [Stock Market EDA](projects/stock-market/analysis.py) | yfinance, Pandas, Matplotlib | 10-year S&P 500 sector volatility and correlation analysis |

## Running a project

```bash
pip install pandas numpy matplotlib seaborn scikit-learn yfinance
python projects/<project-folder>/analysis.py
```

## View Portfolio

Open `index.html` in any browser, or host via GitHub Pages.
